Vi kan dele formularer mellem installationer. Der bliver delt

  1. Elementer og deres indstillinger tilhørende formularen
  2. Titel
  3. Beskrivelse
  4. Kategori



Det vil sige at du kan have særskilte handlere og indstillinger i formularen på de enkelte installationer.
