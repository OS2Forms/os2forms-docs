Der er lavet en statusrapport, hvor du kan se alle vigtige informationer om hvilken installation du kører på og hvor opdateret den er.

Trin | Handling  
---|---  
1 | Klik Rapporter => Statusrapporter  
  
[site] /da/admin/reports/status   
2 |  Her får du indblik i bl.a. PHP-version  
Drupal-version  
Database-version  
og om CRON kører  
  
3 | Du kan også se fejl og advarsler som er på din installation.  
  
Det er yderligere muligt at se detaljer om fx hvilket git-repository som det er på og hvilke specifikke moduler som bør opdateres.
