##  CSS styling af elementer 

Hvis man ønsker en baggrundsfarve på een af beholderne, kan dette meget let angives ved hjælp af CSS. Det gøres ved at klikke et element > Avanceret > Element CSS style

Her tilføjes "background:#f6fef6" - hvilket giver baggrundsfarven lys pastelgrøn.
